# AI-Native Computer Vision Workshop: Manufacturing Quality Control

This repository mirrors the structure of the reference workshop and contains materials for a workshop on probabilistic object detection using YOLO for manufacturing quality control.

## Contents
- `README.md` — this file
- `Solution architecture.png` — conceptual architecture diagram
- `workshop.ipynb` — main workshop notebook
- `workshop_updated_ai_native.ipynb` — updated workshop notebook

## Getting Started
1. Install prerequisites:

```bash
pip install ultralytics opencv-python matplotlib torch torchvision
```

2. Open `workshop.ipynb` with Jupyter Lab or Notebook and follow the sections.

## Prerequisites
- Python 3.8+
- PyTorch 1.8+
- ultralytics (YOLOv8)
- OpenCV
- Matplotlib
